Tableau Course Modules

Module 1: Introduction to Tableau

This module provides an overview of Tableau, its features, and how it is used for data visualization. Topics covered include understanding the Tableau interface, navigating workbooks and sheets, and basic data connections.

Module 2: Getting Started with Tableau

In this module, students will learn how to install Tableau, navigate the interface, and connect to data sources. Topics covered include downloading and installing Tableau Desktop, connecting to different data sources such as Excel and databases, and basic data manipulation.

Module 3: Data Visualization Basics

Students will learn the fundamentals of data visualization, including how to create basic charts and graphs in Tableau. Topics covered include understanding chart types, customizing visualizations, and best practices for effective data visualization.

Module 4: Advanced Data Visualization

This module covers advanced visualization techniques, including interactive dashboards, story points, and advanced chart types. Topics covered include creating interactive dashboards with multiple visualizations, using story points to create narrative-driven presentations, and advanced chart types such as heat maps and box plots.

Module 5: Data Analysis with Tableau

Students will learn how to perform data analysis using Tableau, including filtering, sorting, and grouping data. Topics covered include filtering data to focus on specific subsets, sorting data to identify trends, and grouping data to create meaningful categories.

Module 6: Advanced Tableau Features

In this module, students will explore advanced features of Tableau, such as calculated fields, parameters, and table calculations. Topics covered include creating calculated fields to perform custom calculations, using parameters to make visualizations interactive, and performing advanced calculations using table calculations.

Module 7: Tableau Server and Sharing

This module covers Tableau Server, including how to publish workbooks, create user permissions, and share visualizations with others. Topics covered include publishing workbooks and data sources to Tableau Server, managing permissions to control access to content, and sharing visualizations using embedded views and web authoring.
